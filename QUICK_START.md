## Image Specifications

- **Format**: JPG, PNG, GIF, WebP
- **Size Limit**: 2 MB per image
- **Maximum Images**: 5 per product
- **Recommended Size**: 1000x1000px square
- **Aspect Ratio**: Any ratio works (will resize to fit)

---

## Troubleshooting

**Images don't show?**
- Check file size (must be under 2 MB)
- Ensure file is a valid image format
- Refresh the page

**Can't upload multiple?**
- Hold Ctrl (or Cmd on Mac) to select multiple files
- You can select up to 5 at once

**Image preview looks weird?**
- This is normal for non-square images
- They'll display correctly in the detail page

**Thumbnails not showing?**
- You need to have at least 1 image uploaded
- Make sure you clicked "Publish" after upload
